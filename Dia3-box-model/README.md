## Selectores CSS

Para seleccionar un grupo de elementos,podemos usar el atributo de clase, y el selector "."
Para seleccionar un unico item, podemos usar el atributo de id, y el selector "#"

```CSS

    div {
    color: blue;
    }
    .alumnos {
    color: red;
    }
````
